# AI Software Product Manager (SPM Agent)

An agentic AI system designed to help solo developers plan, execute, and track software projects intelligently.

## Tech Stack

- **Frontend**: Next.js (App Router), Tailwind CSS
- **Backend**: FastAPI (Python)
- **Database & Auth**: Supabase
- **Language Models**: (Planned integration)

## Structure

- `backend/`: FastAPI application
- `frontend/`: Next.js application
- `docs/`: Project documentation and guides
